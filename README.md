## Local Development

To compile, install dependencies, and run the server locally without warnings, follow these steps:

1. Ensure you have Ruby and Bundler installed.
2. Install the necessary dependencies:
   ```bash
   bundle install
   ```
3. Execute the local server:
   ```bash
   bundle exec jekyll serve
   ```
The site will be available at `http://localhost:4000`.
